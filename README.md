# Currency Converter
A simple Python app to quickly convert between currencies.
Currency Converter is a simple Python application that allows users to convert amounts between different currencies quickly and easily. 
It supports major global currencies and provides an intuitive interface for fast conversions.  
This project is a great way to practice Python programming and understand basic financial calculations.

# Usage
1. Clone or download the repository.
2. Run the program.
3. Follow on-screen instructions.
